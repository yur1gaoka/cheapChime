# バスチャイム生成ツール

ブラウザだけで動作する、単一ファイル構成チャイム生成ツールです。  
サーバ不要で `index.html` をそのまま開くだけで使えます。

## 特徴

- `index.html` 単体で完結
- 外部ライブラリ不使用
- Web Audio API によるベル系チャイム合成
- 3点チャイム / 4点チャイムを切り替え可能
- 3種類の音色プリセットを搭載
- 音の高さ、テンポ、音量を調整可能
- `OfflineAudioContext` を使った WAV 書き出し対応

## 使い方

1. `index.html` をブラウザで開きます
2. 「チャイム種類」を選びます
3. 「音色プリセット」を選びます
4. 必要に応じて「音の高さ」「テンポ」「音量」を調整します
5. 「再生」で試聴します
6. 気に入った設定になったら「WAVダウンロード」で保存します

## 初回再生について

ブラウザの仕様上、音声はユーザー操作のあとで有効化されます。  
そのため、最初の再生は必ずボタン操作から開始してください。

## 音作りの方針

このツールは単純なビープ音ではなく、案内音として使えそうな上品なベル感を重視しています。

主な設計要素:

- 基音に加えて 2倍音、3倍音、補助倍音を重ねる
- 各倍音を 2 発のオシレーターで少しだけデチューンして厚みを出す
- アタック直後だけ少し高めから落ち着くピッチ設計にして、打鍵感を出す
- ごく短いトランジェント成分を加えて、最初の一音の印象を強める
- フェードイン / フェードアウト付きの自然なエンベロープでクリックノイズを防ぐ
- 疑似リバーブと短い反射音で、残響感と空間の広がりを足す
- 最終段で軽くコンプレッションをかけ、音量破綻や歪みを抑える

## プリセット

### Standard Bell

標準的なベル感を持つバランス型です。  
最初の一音の存在感と、公共交通らしい落ち着きの両立を狙っています。

### Soft Chime

柔らかく耳当たりの良い音色です。  
余韻をやや長めに取り、案内音としての上品さを強めています。

### Bright Synth Chime

もっとも華やかなプリセットです。  
アタックと倍音の密度をやや強め、最初の一音で印象に残りやすくしています。

## 実装の見どころ

### 1. 設定取得

`getSettings()` で UI の値をまとめて取得します。  
再生処理と書き出し処理の両方が同じ設定を使います。

### 2. 音源本体

`scheduleVoice()` が各ノートのベル本体を生成します。  
倍音ごとにオシレーターを重ね、デチューン、フィルタ、サチュレーション、エンベロープを適用しています。

### 3. アタック成分

`scheduleTransient()` が短いノイズ成分とクリック感のある補助音を加えます。  
これにより、最初の立ち上がりが明瞭になります。

### 4. 残響

`createReverbNetwork()` が短い反射音とフィードバックを組み合わせ、疑似的なリバーブ空間を作ります。

### 5. 再生と書き出し

`renderChimeToContext()` に音源ロジックを集約しています。  
通常再生では `AudioContext`、書き出しでは `OfflineAudioContext` を使い、同じ音作りを共有します。
