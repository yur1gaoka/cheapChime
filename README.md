# バスチャイム生成ツール

ブラウザだけで動作する、単一ファイル構成のチャイム生成ツールです。  
サーバ不要で `index.html` をそのまま開くだけで使えます。

## 特徴

- `index.html` 単体で完結
- 外部ライブラリ不使用
- Web Audio API による案内音向けチャイム合成
- 2点 / 3点 / 4点チャイムを切り替え可能
- 10種類の音色プリセットを搭載
- コード進行プリセットと各ステップごとのコード指定に対応
- 単音 / 2ボイス / 3ボイス / 4ボイスを切り替え可能
- 基準ピッチ、テンポ、音量、リバーブ、Low EQ、High EQ を調整可能
- ループ再生対応
- `OfflineAudioContext` を使った WAV 書き出し対応

## 使い方

1. `index.html` をブラウザで開きます
2. 「チャイム種類」と「音色プリセット」を選びます
3. 必要に応じてコード進行、オクターブ、ボイス数を調整します
4. ピッチ、テンポ、音量、リバーブ、EQ を調整します
5. 「再生」で試聴します
6. 必要なら「ループ」を有効にします
7. 気に入った設定になったら「WAVダウンロード」で保存します

## 初回再生について

ブラウザの仕様上、音声はユーザー操作のあとで有効化されます。  
そのため、最初の再生は必ずボタン操作から開始してください。

## 音作りの方針

このツールは単純なビープ音ではなく、公共交通や案内音として使えそうな上品なベル感を重視しています。

主な設計要素:

- 基音だけで終わらせず、倍音を重ねて厚みを作る
- デチューンは控えめにして、不協和になりすぎない範囲に収める
- アタック直後の印象を整えるために短いトランジェント成分を加える
- フェード付きエンベロープでクリックノイズを防ぐ
- 疑似リバーブと短い反射音で空間の広がりを足す
- 最終段で軽く整音して、音量破綻や歪みを抑える
- 再生処理と書き出し処理で同じ音源ロジックを共有する

## 実装の見どころ

### 設定取得

`getSettings()` で UI の値をまとめて取得します。  
再生処理と書き出し処理の両方が同じ設定を使います。

### 発音計画

`buildNotePlan()` がコード進行、ボイス数、オクターブ設定から発音内容を組み立てます。

### 音源本体

`scheduleVoice()` が各ノートのベル本体を生成します。  
倍音ごとにオシレーターを重ね、デチューン、フィルタ、サチュレーション、エンベロープを適用しています。

### アタック成分

`scheduleTransient()` が短いノイズ成分と補助音を加え、立ち上がりを明瞭にします。

### 残響

`createReverbNetwork()` が短い反射音とフィードバックを組み合わせ、疑似的なリバーブ空間を作ります。

### 再生と書き出し

`renderChimeToContext()` に音源ロジックを集約しています。  
通常再生では `AudioContext`、書き出しでは `OfflineAudioContext` を使い、同じ音作りを共有します。
